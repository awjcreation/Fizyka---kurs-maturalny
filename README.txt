Elektrostatyka – PWA

Zawartość:
- index.html — ekran startowy aplikacji
- course.html — kurs
- practice.html — zadania krok po kroku
- quiz.html — quiz
- manifest.webmanifest
- service-worker.js
- icon-192.svg, icon-512.svg

Jak uruchomić:
1. Rozpakuj folder.
2. Wgraj go na prosty hosting statyczny albo otwórz przez lokalny serwer WWW.
   PWA nie zainstaluje się poprawnie przy zwykłym otwarciu pliku z dysku przez file://
3. Otwórz index.html przez adres http(s).
4. W przeglądarce mobilnej wybierz „Dodaj do ekranu głównego” albo użyj przycisku „Zainstaluj aplikację”.

Uwaga:
- Sama zawartość działa offline po pierwszym załadowaniu przez service worker.
- Do pełnego działania PWA potrzebny jest serwer HTTP lub HTTPS.
